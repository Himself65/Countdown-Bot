# Countdown-Bot

#### 项目介绍

QQ群倒计时bot.

#### 软件架构

本程序为后端.

前端为Coolq.

#### 安装教程

前置需求:

- Coolq
- Python3.5+
- cqhttp (使用pip安装)

#### 使用说明

- 运行方式:使用python运行main.py

- 可以在config.py修改配置

- 在有该bot的群里输入/help查看帮助
